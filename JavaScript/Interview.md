### Что такое IIFE?

`IIFE (Immediately Invoked Function Expression)` — это конструкция, позволяющая вызывать функцию непосредственно после ее определения.  
Чаще всего она используется, чтобы не допустить загрязнения глобального пространства имён. Переменные, объявленные в IIFE, невидимы за пределами этой функции.

#### Именованная IIFE

- Это **классическая IIFE** (Immediately Invoked Function Expression).
- Используется **именованная функция** (`IIFE`), которая вызывается сразу после объявления.
- Работает во всех версиях JavaScript.
- Имя функции (`IIFE`) доступно только внутри самой функции (полезно для рекурсии).
  
```js
(function IIFE() {
    console.log("IIFE");
})();
```

#### Cтрелочная IIFE

- Не имеет собственного `this`, `arguments`, `super`, `new.target`.
- Не может быть именованной (стрелочные функции всегда анонимны).
  
```js
(() => (console.log("IIFE"))
```

Если нужно просто выполнить код, можно использовать стрелочную IIFE.
Если нужна рекурсия или более сложная логика, лучше классическая IIFE.

#### Что выведет следующий код?

```js
const output = (function(x) {
	delete x;
	return x;
})(0); // может передаваться аргумент

console.log(output);
```

Результат кода — **0**. Оператор **delete** используется для удаления свойств объектов, а **x** не является свойством объекта. **х** — это локальная переменная. Оператор **delete** не воздействует на локальные переменные.
